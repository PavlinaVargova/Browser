DevTools

DevTools sada nástrojů, která umožňuje více prozkoumat sránky z pohledu zdrojového kódu. Umožňuje zjistit, jak je kód napsán, jaké jsou použité značky, označní ID, class (zjištěné údaje lze použít u automatického testování), jaké cookies stránka používá, zda má obrázek validní URL apod.

Jak získat zdrojový kód html stránky pomocí nástroje Postman

Tento návod předpokládá, že máte již u sebe Postmana nainstalovaného (ať už jako deskopovou variantu nebo jako doplněk do Chromu).

zdroj obrázku:guru99

  1. Spuste Postmana. Mělo by se vám zobrazit podobné okno jako je na obrázku výše.
  2. Vedle příkazu GET do kolonky "Enter request URL" vložte url adresu stránky, u níž chcete zjistit html kód. Pokudmáte jiný příkaz místo GET, změňte ho.
  3. Následně stačí pouze odeslat svůj dotaz. Požadavek se odešle a vrátí se nám odpověď v podobě html kódu.

Zkoušela jsem také jiné příkazy kromě GET a u příkazu VIEW se mi html zdrojový kód ukázal stejně.

Rozdíl mezi XML a JSON

Oba soubory nesou informace mezi aplikacemi. Rozdíl spočívá v jejich složitosti.

XML

XML má přesně danou stromovou strukturu s jedním kořenem. Vyznačuje se párovými značkami. Používá se všude tam, kde je nutné definovat struktura informací, např. v bankovnicví - zadání bankovního příkazu, kdy proběhne kontrola, zda soubor obsahuje všechny potřebné informace - validační struktura.

JSON

Není tak obsáhlý jako XML, nemá validační strukturu. Je podporován napříč různými programovacími jazyky. Používá se např. v eshopech .
